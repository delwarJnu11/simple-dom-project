Discuss the scope of var, let, and const!

var: Var is a keyword we use it to declare a variable. var is works with global and local both. when we declare a varibale with var keyword inside a function block. we can get value the of this variable outside the function. as well as, when we declare a variable with var keyword in global scope the we acess the variable anywhere from the js file. Var can be updated and re-declared and It can be declared without initialization.

let: Let is a keyword we use it to declare a variable. let is a block scope. when we declare a varibale with let keyword inside a block. we can not get value the of this variable outside the function. Let can be updated but re-declared is not possible. Let can declare without initialize value.

const: const is a keyword we use it to declare a variable. const is a block scope. when we declare a varibale with const keyword inside a block. we can not get value the of this variable outside the function. const can not be updated or re-declared. const can not be declared without initialize value.

What do you mean by REST API?

RestApi: A REST API (also known as RESTful API) is an application programming interface (API or web API) that conforms to the constraints of REST architectural style and allows for interaction with RESTful web services. REST stands for representational state transfer and was created by computer scientist Roy Fielding.
